Core Implementation Details

Vanilla SPA Engine

Приложение построено на Pure JS (ES6+) без фреймворков. Модульная архитектура обеспечивает полный контроль над логикой и высокую производительность. Состояние данных (State) управляется вручную

Browser API Routing

Навигация реализована через URLSearchParams. Это легкое решение для SPA, которое позволяет сохранять состояние страницы в URL и мгновенно переключаться между разделами без перезагрузки

Dynamic Appearance

Динамическая смена тем на CSS-переменных. Выбор пользователя сохраняется в LocalStorage, обеспечивая консистентный дизайн и удобство при каждом входе в систему
